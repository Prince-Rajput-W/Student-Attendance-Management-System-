# 📘 Student Attendance Management System

## 📌 Description

The **Student Attendance Management System** is a console-based
application developed in C that helps manage student records and track
attendance efficiently. The system replaces manual attendance methods
with a digital solution using file handling for persistent data storage.

------------------------------------------------------------------------

## 🚀 Features

-   Add new students with unique roll numbers
-   Prevent duplicate entries
-   View all student records
-   Mark attendance (Present/Absent)
-   View attendance report
-   Data stored permanently using file handling

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   **Language:** C
-   **Concepts:** Structures, File Handling, Functions, Menu-driven
    Programming

------------------------------------------------------------------------

## 📂 File Structure

attendance.c → Main source code
students.dat → Data file (created automatically)

------------------------------------------------------------------------

## ▶️ How to Run

### Step 1: Compile the code

gcc attendance.c -o attendance

### Step 2: Run the program

./attendance

------------------------------------------------------------------------

## 📋 Menu Options

1.  Add Student
2.  View Students
3.  Mark Attendance
4.  View Attendance
5.  Exit

------------------------------------------------------------------------

## 💡 How It Works

-   Each student has:
    -   Roll Number (Unique)
    -   Name
    -   Attendance Status
-   Data is stored in a binary file (`students.dat`)
-   Attendance can be updated anytime

------------------------------------------------------------------------

## 🎯 Objective

To develop a simple and efficient system that automates student
attendance management using basic C programming concepts.
